# QGIS Attribute Table to Word Document Automation

This project automates the extraction of attribute table data from QGIS
and organizes the information into a structured Microsoft Word document.

## Overview

In many GIS workflows, attribute data from spatial layers needs to be
reported in a structured document format. This script uses Python within
QGIS to read attribute table data and generate a formatted Word document
automatically.

## Features

- Extracts attribute table information from a QGIS layer
- Organizes the data in a structured format
- Automatically generates a Word (.docx) document
- Reduces manual data entry for reporting

## Technologies Used

- Python
- QGIS Python API (PyQGIS)
- python-docx

## Output

An example output document is included in this repository.
